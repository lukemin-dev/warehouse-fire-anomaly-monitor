# Hardware

## 사용 부품

| 부품 | 역할 |
| --- | --- |
| Raspberry Pi 4 | 센서 수집 및 서버 전송 |
| MQ-2 | 가스/연기 감지 |
| SEN040132 | 불꽃 감지 |
| MCP3008 | 아날로그 값을 SPI로 변환 |
