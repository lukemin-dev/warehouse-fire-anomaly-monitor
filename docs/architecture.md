## 데이터 흐름

```
센서 (MQ-2, SEN040132)
    │
    │ DO (0/1) → GPIO 직접 연결
    │ AO (0~1023) → MCP3008 → SPI
    ▼
라즈베리파이 sensor_test.py
    │
    │ HTTP POST /api/event (2초 주기)
    │ { mq2_raw, mq2_status, flame_raw, flame_status, mq2_ao, flame_ao }
    ▼
AWS EC2 Flask 서버
    │
    ├─ SQLite DB 저장
    ├─ IsolationForest AI 판정
    └─ 경고 생성 (FIRE / FLAME / GAS_SMOKE / ANOMALY)
         │
         ├─ 모바일 앱 (3초 폴링)
         └─ 웹 대시보드 (3초 자동 갱신)
```

## 센서 데이터 기준

### MQ-2 아날로그 값 (AO)

| 구간 | AO 값 | DO 값 | AI 판정 |
|------|-------|-------|---------|
| 정상 | 0 ~ 30 | 1 | 정상 |
| 전조 | 30 ~ 200 | 1 | AI 이상 |
| 감지 | 200 이상 | 0 | 경고 + AI 이상 |

### 실측 기준값

| 테스트 | MQ-2 AO 최대값 |
|--------|----------------|
| 정상 (공기) | 10 ~ 15 |
| 알코올 근접 | 147 ~ 217 |
| 라이터 가스 | ~ 900 |

### 불꽃 센서 아날로그 값 (AO)

| 상태 | AO 값 |
|------|-------|
| 정상 | 810 ~ 812 |
| 감지 | 낮아짐 (확인 예정) |

## 경고 유형

| 유형 | 조건 | 색상 |
|------|------|------|
| FIRE | MQ-2 감지됨 + 불꽃 감지됨 동시 | 빨강 |
| FLAME | 불꽃만 감지됨 | 빨강 |
| GAS_SMOKE | MQ-2만 감지됨 | 주황 |
| ANOMALY | 센서 정상이나 AI 이상 패턴 감지 | 보라 |

## API 엔드포인트

| 메서드 | 경로 | 설명 |
|--------|------|------|
| POST | /api/event | 라즈베리파이 센서 데이터 수신 |
| GET | /api/latest | 최신 센서값 1건 |
| GET | /api/readings | 최근 100건 |

## 하드웨어 구성

| 부품 | 역할 | 연결 |
|------|------|------|
| 라즈베리파이 4 | 메인 컨트롤러 | - |
| MQ-2 | 가스/연기 감지 | DO → GPIO17, AO → MCP3008 CH0 |
| SEN040132 | 불꽃 감지 | DO → GPIO27, AO → MCP3008 CH1 |
| MCP3008 | ADC 변환 (아날로그→디지털) | SPI (GPIO8,9,10,11) |
