## 1. 센서 배선 및 환경 구성

- 라즈베리파이 4 부팅 및 GPIO 배선 완료
- MQ-2 가스/연기 센서 연결 (VCC: 5V, DO: GPIO)
- SEN040132 불꽃 감지 센서 연결 (VCC: 3.3V, DO: GPIO)

---

## 2. 센서 테스트 코드

```python
import RPi.GPIO as GPIO
import time

# GPIO 핀 번호 (BCM 기준)
MQ2_PIN   = 17  # MQ-2 가스/연기 센서 DO
FLAME_PIN = 27  # SEN040132 불꽃 감지 센서 DO

GPIO.setmode(GPIO.BCM)
GPIO.setwarnings(False)
GPIO.setup(MQ2_PIN,   GPIO.IN)
GPIO.setup(FLAME_PIN, GPIO.IN)

print("센서 모니터링 시작 (Ctrl+C 종료)")
print(f"MQ-2 핀: GPIO{MQ2_PIN} | 불꽃 센서 핀: GPIO{FLAME_PIN}")
print("-" * 50)

try:
    while True:
        mq2_raw   = GPIO.input(MQ2_PIN)
        flame_raw = GPIO.input(FLAME_PIN)

        # DO 출력: 감지 시 LOW(0), 정상 시 HIGH(1)
        mq2_status   = "감지됨" if mq2_raw   == GPIO.LOW else "정상"
        flame_status = "감지됨" if flame_raw == GPIO.LOW else "정상"

        print(f"[MQ-2 가스/연기] {mq2_status} (RAW={mq2_raw}) | "
              f"[불꽃] {flame_status} (RAW={flame_raw})")

        time.sleep(1)

except KeyboardInterrupt:
    print("\n종료")
finally:
    GPIO.cleanup()
```

## 5. EC2 서버 구축 및 데이터 수신

### 서버 환경

- AWS EC2 t3.micro / Ubuntu 22.04 LTS
- Python Flask 서버 운영 중
- 퍼블릭 IP: `<EC2_PUBLIC_IP>` / 포트: 5000

### EC2 서버 수신 코드 (`app.py`)

```python
from flask import Flask, request
from datetime import datetime
import json

app = Flask(__name__)
events = []

@app.route('/api/event', methods=['POST'])
def receive_event():
    data = request.get_json()
    data['received_at'] = datetime.utcnow().isoformat()
    events.append(data)
    print(f"[수신] {data}")
    return json.dumps({"status": "ok"}, ensure_ascii=False), 200

@app.route('/api/events', methods=['GET'])
def get_events():
    return app.response_class(
        response=json.dumps(events, ensure_ascii=False, indent=2),
        status=200,
        mimetype='application/json; charset=utf-8'
    )

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=5000)
```

### 라즈베리파이 → EC2 전송 흐름

```
라즈베리파이 (GPIO 센서 읽기)
    │
    │ POST /api/event (2초 주기)
    │ JSON 페이로드: ts, mq2_raw, mq2_status, flame_raw, flame_status
    ▼
AWS EC2 Flask 서버 (`<EC2_PUBLIC_IP>:5000`)
    │
    ├─ /api/event  → 수신 및 메모리 저장
    └─ /api/events → 저장된 전체 데이터 조회
```
